\section{Exercício 1}

\begin{itemize}
\item Faça a carga do dataset iris.
\item Exclua o atributo Species do dataset.
\item Calcule a correlação entre os atributos restantes.
\item Imprima gráficos que demonstram a correlação positiva alta no caso de
  alguns atributos.
\item Imprima gráficos que demonstram a não correlação entre atributos.
\item Calcule o desvio padrão dos atributos. Para aquele atributo com
  maior desvio padrão e para aquele atributo com menor desvio padrão
  imprima gráficos de histograma.
\item Teste o comando summary(iris). O que ele retorna? Comente o resultado.
\end{itemize}

\section{Exercício 3}

\begin{itemize}
\item Faça o download do arquivo que está em http://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all\_month.csv
\item Usando o comando read.csv do R\footnote{Se você tiver dúvidas
    sobre o uso desta função então digite help(read.csv)}, importe as
  informações do arquivo all\_month.csv em um data.frame.
\item Entenda melhor o significado de cada atributo acessando o site
  http://earthquake.usgs.gov/earthquakes/feed/v1.0/csv.php.
\item Usando a função 

\begin{alltt}
diaHorario <- date()
\end{alltt}

armazene o dia e horário quando os dados foram coletados.

\item Qual a quantidade de exemplos (abalos) foi coletada?

\item Quantos exemplos existem no dataset?
 
\item Quais são os tipos de cada atributo?

\item O R conseguiu importar corretamente todos os atributos com os

\item Quantos tipos distintos de abalos existem no dataset?

\item Existe alguma correlação entre a profundidade do abalo sísmico

\end{itemize}

\section{Desafios}

\begin{itemize}
\item O dataset com abalos sísmicos possui um atributo chamado
  \textit{time}. Como o próprio nome diz, este atributo é um atributo
  do tipo \textit{time}. No entanto, ele foi importado como
  \textit{factor} - o que está errado. Converta os valores deste
  atributo para o tipo correto. Depois de convertido, tente utilizar
  esta informação criando plots com séries temporais.

\item Cada registro possui uma latitude e uma longitude. Portanto, é
  possível plotar as informações dos abalos sísmicos em um mapa. Que
  tal tentar?

\end{itemize}
