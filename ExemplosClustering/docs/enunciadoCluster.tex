\section{Questões sobre cluster plano}

\begin{enumerate}
\item Utilizando o dataset \textbf{ruspini} do pacote
  \textbf{cluster}, execute uma análise de cluster utilizando o
  algoritmo \textbf{k-means}.
\item Faça uma análise de cluster utilizando o dataset sobre
  \textbf{abalos sísmicos} do exercício anterior. Considere apenas as
  variáveis de \textbf{longitude} e \textbf{latitude}. 
\item Imprima a informação dos clusters em um mapa georeferenciado - o
  mesmo utilizado no exercício anterior.
\item Faça uma análise de cluster utilizando o dataset
  \textbf{survey}, atributos \textbf{Age} e \textbf{Height},
  do pacote \textbf{UsingR}. Implemente três cenários diferentes: 
\begin{enumerate}
\item Com os valores originais;
\item Com os valores da altura (Height) em metros, e;
\item Com os valores da altura (Height) e idade (Age) devidamente
  normalizados. 
\end{enumerate} 

Comente o que acontece em cada um dos casos.

\item Faça uma análise de cluster utilizando o dataset
  \textbf{survey}, atributos \textbf{Wr.Hnd}, \textbf{NW.Hnd}, \textbf{Age} e \textbf{Height},
  do pacote \textbf{UsingR}. Faça o pré-processamento adequado dos
  dados e utilize a função \textbf{heatmap} para auxiliá-lo na sua análise. 

\item Utilizando o dataset \textbf{survey}, atributos \textbf{Exer}
  (sobre o hábito de fazer exercícios) e \textbf{Smoke} (sobre o
  hábito de fumar), faça uma análise de cluster deste dataset. 

\item Levando-se em consideração o dataset do item anterior, inclua o
  atributo de sexo (\textbf{Sex}) e faça a mesma análise.

\item Faça a carga do dataset \textbf{Animals} do pacote \textbf{MASS}
  e faça uma análise de cluster considerando os dados que estão no
  dataset \textbf{Animals}. Responda as seguites perguntas:

\begin{itemize}
\item Que método você utilizou para determinar o número de clusters ideal?
\item Houve algum tipo de pré-processamento? Em caso positivo, qual
  foi a razão para este pré-processamento?
\item Qual é o número de clusters ideal que você encontrou?
\item Faça uma descrição mais alto nível para cada cluster
  encontrado. Experimente rotular os agrupamentos.
\end{itemize}

\end{enumerate}

\section{Questões sobre cluster hierárquico}

\begin{enumerate}

\item Faça uma análise de cluster hierárquico utilizando o dataset
  \textbf{survey}, atributos \textbf{Wr.Hnd}, \textbf{NW.Hnd}, \textbf{Age} e \textbf{Height},
  do pacote \textbf{UsingR}.

\item Em determinadas situações é difícil identificar o número
  adequado de clusters utilizando o método \textit{elbow}. Outro
  método que pode ser utilizado para identificar um número adequado de
  clusters planos é o próprio cluster hierárquico. Neste caso, faça
  uma análise com cluster hierárquico utilizando o dataset sobre
  \textbf{abalos sísmicos}. Considere apenas as variáveis de
  \textbf{longitude} e \textbf{latitude}.  

\end{enumerate}

\section{Códigos que podem ser úteis}

\begin{alltt}
elbow <- function(dataset)\{
  wss <- numeric(15)
  for (i in 1:15) 
    wss[i] <- sum(kmeans(dataset,centers=i, 
                  nstart=100)$withinss)
  plot(1:15, wss, type="b", main="Elbow method", 
       xlab="Number of Clusters",
       ylab="Within groups sum of squares", 
       pch=8, col="red")
\}
\end{alltt}
